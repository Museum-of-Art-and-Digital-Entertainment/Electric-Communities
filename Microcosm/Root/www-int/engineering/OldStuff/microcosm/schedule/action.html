Microcosm Engineering Action Items

These represent things we need to do immediately in order get on course to deliver Microcosm

  1. Assign or eliminate unscheduled tasks in the Engineering Task List.

  2. Identify hard deliverables with dates for Engineering to hit. This should be enumerated in Engineering Commitments. We need to identify any vacation/etc. time for engineers to make sure time allocated is consistent with the deliverables.

  3. Determine how to best balance the need for meetings with the need for cranking on implementation.

  4. Identify deliverable each task is for, and indicate that in the Engineering Task List. If a task spans multiple deliverables, identify each one.

  5. Cross correlate the Engineering Task List with the Microcosm Master List.

  6. Identify "engineered tasks" which are not currently accounted for and may be designed or implemented by other groups (such as distribution, mastering of media, operations, etc.)

  7. Prioritize Engineering tasks, and make sure they are done in the most logical order to prevent bottlenecks.

  8. Identify highest priority issues preventing us from getting Microcosm done.

  9. Meet and resolve notes (issues actually) in Engineering Task List.
