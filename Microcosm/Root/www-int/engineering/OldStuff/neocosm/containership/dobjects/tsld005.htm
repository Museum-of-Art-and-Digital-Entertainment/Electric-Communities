‘D’

  • Not language…rather is just library of (~50) java classes

  • *Minimal* distributed object system

    • Generically supports many-to-many connectivity
    • Asynchronous messaging (within and between process)
    • Principle of Least Authority security
    • Principle of Least Commitment approach to most everything else
      • Small, fast, very exstensible
      • All Java

  • Terminology: Unum == ‘Dobject’ (for Distributed Object)
