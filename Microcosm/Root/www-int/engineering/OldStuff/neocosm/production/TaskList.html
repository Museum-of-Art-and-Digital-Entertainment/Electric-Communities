Overview

This is the task list generated in February. John and I had just started to work on the first items when the plug was pulled.

This list, as is, amounts to about 48 engineering-weeks of effort. It will be critical to decide what the right set of tools to do is because doing a complete set is a very large piece of work.

We need input to do the right work here.
