Step 8. Build Unum Implementation

As mentioned earler, an unum is a mere conceptualization and at programming level consists of its presences. Thus, an unum implementation simply assigns presences’ implementation to all presence roles in the unum structure. The associated unum structure is included, and the presence statements associate a name to each presence implementation.


	// ----- Build unum implementation -----
	export unum impl uiPeekablePokeable {
	  structure usPeekablePokeable;
	  resence upnHostPeekablePokeable impl piHostPeekablePokeable;
	  presence upnClientPeekablePokeable impl piClientPeekablePokeable;
	}
