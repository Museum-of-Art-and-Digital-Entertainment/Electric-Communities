Kawa Notes

How to use Kawa with E.

  1. Install Kawa [get it here].
  2. Do a "cvs up -d ec" in /src/ec4/javasrc.
  3. Run Kawa from your Start Menu.
  4. It'll open up a dialog that asks you to set the directories in which to find the JDK. Set them to whatever JDK installation you'd like to use.
  5. Set the java source directory to /src/ec4/javasrc.
  6. Create a new project.
  7. Use the ADD FILES menu to add all java files from the following directories.
    1. /src/ec4/javasrc/ec/e/evm
    2. /src/ec4/javasrc/ec/e/scope
    3. /src/ec4/javasrc/ec/e/syntax
    4. /src/ec4/javasrc/ec/elib/base
    5. /src/ec4/javasrc/ec/elib/prim
    6. /src/ec4/javasrc/ec/elib/quasi
    7. /src/ec4/javasrc/ec/tables
    8. /src/ec4/javasrc/ec/trace
    9. /src/ec4/javasrc/ec/util
    10. /src/ec4/javasrc/ec/util/assertion
    11. /src/ec4/javasrc/ec/vcache
  8. Under the Project menu, select Interpreter Options..., select the Java Class Name to Run tab and enter ec.e.syntax.EMain
  9. Under the Project menu, or in the environment before starting Kawa, set your CLASSPATH to include /src/ec4/javasrc.
  10. Rebuild All
  11. Under the Build menu, select Run, Debug/Run, or JDB Debug..
Note that I had to add the compiler option -J-mx32m to get the 1.2 compiler to compile all of Elib. It runs out of memory if you don't set that. You can do this from the Project/Compiler Options... menu.
