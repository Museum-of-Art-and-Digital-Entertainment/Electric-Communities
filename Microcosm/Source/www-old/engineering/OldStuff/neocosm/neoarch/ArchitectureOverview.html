Introduction

This document is a high-level architecture overview for the Neocosm system.  The intention is to provide a clear overall view of the Neocosm system architecture, and provide links for more detailed subsystem requirements, architecture and design documents, API definitions, and subsystem source code.   This document, where possible, will reference these external documents as they become available.  These documents will a) give details on all of the subsystems; b) identify owners for each subsystem; c) can be used to track requirements, architecture and design, and implementation progress.
