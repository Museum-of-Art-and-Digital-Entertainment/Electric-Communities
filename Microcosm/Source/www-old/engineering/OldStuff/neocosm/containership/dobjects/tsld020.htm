Secure ‘meta interfaces’ for capability requests

  • DObject requests to access local system

    • View.getViewFacet(String requestName, Object data)
    • Intended for all UI interface access, all other I/O, all other network stacks, etc.

  • DObject requests to access *other DObjects * (asynch)

    • View.requestDObjectCapability(otherId, String requestName, Object data)
    • Uses P.O. Box pattern for capabilities exchange
