Dobject gives facet to View (DObjectViewFacet)

  • View can send notifications

    • Asynchronous
    • Group membership changes
      • DObject arrival/departure

  • View can kill client presences at any time

  • Abstract Dobject class guarantees functionality for untrusted code

    • Classloader guarantees Dobject class loaded locally
    • Dobject class defines key methods as *final*
      • Id handling
      • Key creation/destruction notifications
      • Message processing
    • View can depend upon behavior of key methods for construction, destruction, etc.
