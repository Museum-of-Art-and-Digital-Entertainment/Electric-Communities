If you have to have an unum reference, and you have client and host versions of your ingredient, then make sure only the host keeps an actual reference. The client should request it from the host when it needs it. You can use channels to make this process optimistic, e.g:

  // Use code like this in any client ingredient method.
  // The client init contains no reference to the Unum.
  Unum blah;
  environment.basePresence <- pGetReference(&blah);
  blah <- dosomething();
