Aria: A Keep-It-Simple Solution

In the spirit of keeping things simple and avoiding the creation of new purple boxes a reasonable solution, although no elegant, is to have a default exception handling mechanism to which all exceptions from lines of computation are thrown. The details of such mechanism will be left up to the designer. The code below behaves that way. 

	eclass Foo {
	  int myID;
	  Bud myLastBud;
	  Bud myDefaultBud = defaultBud;
	  method uFoo(Bar bar, Bud bud) {
	    myID++;
	    int oldID = myID;
	    etry {
	      bar <- uBar(bud);
	    } ecatch {
		  if (oldID == myID) {
	        myLastBud = myDefaultBud;
		  }
	    }
	  }
	}
