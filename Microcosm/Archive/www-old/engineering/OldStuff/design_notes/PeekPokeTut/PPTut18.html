Step 3. Modify RunMe file

The RunMe file to generate the host and three clients is listed here. The difference between this file and the one for only one client lies in the client file generation code.


	#!/bin/sh
	# -v
	# java -debug ec.e.start.EBoot ec.pl.runtime.Agency Agent=ec.pl.examples.PeekPoke
	LSERVER=gracie.communities.com:666
	PID1=test1.$$
	PID2=test2.$$
	PID3=test3.$$
	ZZZ="RegisterWith=$LSERVER SearchPath=$LSERVER"
	set -x
	java ec.e.start.EBoot ec.pl.runtime.Agency Agent=ec.pl.examples.PeekPoke 
		PublisherId=$PID1 $ZZZ > output0 &
	sleep 5
	java ec.e.start.EBoot ec.pl.runtime.Agency Agent=ec.pl.examples.PeekPoke 
		PublisherId=$PID2 $ZZZ url=e://$LSERVER/$PID1/DoPeekPoke > output1 &
	sleep 5
	java ec.e.start.EBoot ec.pl.runtime.Agency Agent=ec.pl.examples.PeekPoke 
		PublisherId=$PID3 $ZZZ url=e://$LSERVER/$PID1/DoPeekPoke > output2 &
	sleep 5
	java ec.e.start.EBoot ec.pl.runtime.Agency Agent=ec.pl.examples.PeekPoke 
		PublisherId=$PID4 $ZZZ url=e://$LSERVER/$PID1/DoPeekPoke > output3
