Step 1: Code for receiver object

In general, the Receiver class contains the methods that define what is actually done with the messages received from sender objects. The task of the receiver object for this tutorial is to print the message received. The code is shown below. canReceive is a method of eclass Receiver that accepts a string message and prints it.

	// ----- Receiver Class -----

	eclass Receiver {
	  emethod canReceive(String message) {
	    System.out.println(message);
	  }
	}
