Here is an example of the consequences of ecatch being an ewhen.


eclass Forwarder
{
  emethod transfer(String message, Receiver r)
    {
      EBoolean itDone;
      etry
        {
          System.out.println("Forwarder transferring " + message);
          r <- receive(message, &itDone);
        }
      ecatch (RtEException x)
        {
          System.out.println("Forwarder: I have caught an exception.");
        }
      loggingReceiver <- log("All is OK: " + message);
    }
}

It doesn't matter if Receiver r throws an exception, the logging message will always be sent to the loggingReceiver.
