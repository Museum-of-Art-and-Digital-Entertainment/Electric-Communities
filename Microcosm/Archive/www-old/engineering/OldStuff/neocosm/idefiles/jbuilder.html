Borland JBuilder Notes

How to use Borland's JBuilder with E.

  1. Install JBuilder [Published as "JBuilder" on Cassandra].
  2. Do a "cvs up -d ec" in /src/ec4/javasrc.
  3. Run JBuilder from your Start Menu (in Borland JBuilder).
  4. Open up a Windows Explorer and navigate to /src/ec4/javasrc/ec/e/evm directory.
  5. Select all the .java files drag and drop them onto the upper left corner of the Project window.
  6. Repeat for the following directories:
    1. /src/ec4/javasrc/ec/e/scope
    2. /src/ec4/javasrc/ec/e/syntax
    3. /src/ec4/javasrc/ec/elib/base
    4. /src/ec4/javasrc/ec/elib/prim
    5. /src/ec4/javasrc/ec/elib/quasi
    6. /src/ec4/javasrc/ec/tables
    7. /src/ec4/javasrc/ec/trace
    8. /src/ec4/javasrc/ec/util
    9. /src/ec4/javasrc/ec/util/assertion
    10. /src/ec4/javasrc/ec/vcache
  7. Under the File menu, select Project properties..., select the Run/Debug tab and set the Default Runnable File to \src\ec4\javasrc\ec\e\syntax\EMain.java.
  8. Build the project (Build menu or ctrl-F9).
  9. Under the Run menu, select Debug "EMain" or Run "EMain".
