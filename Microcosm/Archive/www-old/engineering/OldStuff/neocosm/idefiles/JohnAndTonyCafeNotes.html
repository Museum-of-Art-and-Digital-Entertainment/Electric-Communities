Followup from John 2/27/98:
To make the classes go into our classfiles hierarchy, you've got to specify the output directory in Visual Cafe's Project/options dialog. Choose that menu item, then click the Directory tab, then select "Output files" from the combo box. Here enter the root for our classfiles hierarchy (e.g. "C:\Build\classes"). Now under the Project tab in this same window type in the fully-qualified class name as the Main Class (e.g., "ec.cosm.tools.appearance.Appearance2DEditor").
